Version: v1.6

JavaScript

To install the JavaScript SDK, run:

npm install --save @oryd/hydra-client

Configuration#

Basic configuration#

import { AdminApi } from '@oryd/hydra-client'
// Set this to Hydra's URLconst hydraAdmin = new AdminApi('http://localhost:4445')

API Usage#

hydraAdmin.listOAuth2Clients(10, 0).then(({ body }) => {  body.forEach((client) => {    console.log(client)  })})
